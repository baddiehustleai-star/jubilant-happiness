# Public Assets

This directory contains static assets that are served directly by Vite.

## Logo

To add your custom Photo2Profit logo:

1. Save your logo image as `logo.png` in this directory
2. The logo should be at least 192x192px for best quality
3. PNG format with transparency is recommended for a professional look

The landing page will automatically load `/logo.png` from this directory.

For now, `logo.svg` is available as a placeholder.
