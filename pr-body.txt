## üîê Required GitHub Secrets

Configure these in **Settings ‚Üí Secrets and variables ‚Üí Actions**:
- `GOOGLE_APPLICATION_CREDENTIALS_JSON` - Google Cloud service account (required)
- `FIREBASE_SERVICE_ACCOUNT` - Firebase service account (required)
- `SLACK_WEBHOOK_URL` - Slack notifications (optional)
- `CRON_SECRET` - API cron endpoint authentication (optional)
